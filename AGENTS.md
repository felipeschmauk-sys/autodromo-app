# Idioma de la UI

Todos los textos visibles para el usuario van en **español neutro de Latinoamérica**.
Nunca usar voseo argentino/rioplatense: escribir "Selecciona", "Arrastra", "Escribe",
"puedes", "presiona" — NO "Seleccioná", "Arrastrá", "Escribí", "podés", "tocá".
